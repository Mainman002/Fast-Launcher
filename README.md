# Fast Launcher
An M2 Mac app launcher gui with fuzzy search functionality.

### Features
- Fuzzy search to easily find apps containing spaces and some special characters
- A list of all apps found in the Applications directory
- "Shift + Command + Spacebar" Keyboard command to open the menu
- "Escape" Keyboard command to close window

### How to Build
Open Xcode, hold "Command" and tap "B" to begin building the application. Once completed you should be able to run the newly built FastLauncher.app

### App Debug Directory
```
~/Library/Developer/Xcode/DerivedData/FastLauncher-"Random_String"/Build/Products/Debug/FastLauncher.app
```
